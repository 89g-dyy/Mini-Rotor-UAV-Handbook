                        前言
在开始学习前，我想读者应该先思考一下无人机是什么，可能大多数人会觉得是玩具、工具、亦或者是会飞的相机，但是我要强调，无人机不仅可用于救援、测绘等，同时也是一种携带方便、成本低、易上手的武器，所以各位在学习的时候一定要对无人机有敬畏之心。
即使不加装任何模块，其桨叶的高速转动也有概率取人性命，所以本教材不仅要求学者具有高道德水平，同时还需要学者具有强烈的安全意识，无论何种情况，请优先以自己安全为主。
为了和谐社会发展，读者尝试实践时，一定要抛开设计武器的概念，希望读者谨记。
本教材适合大学希望从事无人机行业的各理工科学生读者，读者将获得明确的技术路线指导，也可选择性的学习部分知识内容，无论怎样，只要认真学习，一定能获得成果。
